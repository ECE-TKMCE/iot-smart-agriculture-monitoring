# 🌱 Enhancing Agricultural Productivity through Advanced Soil Monitoring

---

## 📌 Overview

This project is an IoT-based solution designed to monitor critical soil parameters in real time. This system leverages the ESP32 microcontroller and an RS485-based soil sensor to provide accurate data on soil health, enabling informed agricultural decisions.

The project focuses on delivering a **low-cost, scalable, and efficient monitoring system** suitable for precision agriculture and smart farming applications.

---

## 🎯 Objectives

* Monitor soil parameters in real time
* Enable data-driven agricultural decisions
* Build a low-cost IoT solution for farmers
* Integrate cloud monitoring with a local dashboard

---

## 🚀 Features

* 📡 Real-time soil monitoring
* 🌐 Local web dashboard hosted on ESP32
* ☁️ Cloud integration using ThingSpeak
* 🔌 RS485 Modbus RTU communication
* 📊 Continuous data logging and visualization

---

## 📊 Parameters Monitored

* Soil Moisture
* Soil pH
* Electrical Conductivity (EC)
* Nitrogen (N)
* Phosphorus (P)
* Potassium (K)

---

## 🛠️ Hardware Components

* ESP32 DevKit C
* ZTS 3002 7-in-1 Soil Sensor
* TTL to RS485 Converter Module
* Power Supply Unit
* Connecting Wires

---

## 💻 Software & Technologies

* Arduino IDE
* Embedded C/C++
* HTML, CSS, JavaScript
* ThingSpeak (Cloud Platform)
* Modbus RTU Protocol

---

## ⚙️ System Architecture

1. Soil sensor collects real-time data
2. Data transmitted via RS485 (Modbus RTU)
3. ESP32 processes sensor data
4. Data is:

   * Displayed on local web dashboard
   * Uploaded to ThingSpeak cloud
5. Users monitor data remotely
