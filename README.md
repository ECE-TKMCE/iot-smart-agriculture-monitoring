# Enhancing Agricultural productivity through advanced soil monitoring
📌 Overview

The Smart Soil Monitoring System is an IoT-based solution designed to monitor critical soil parameters in real time. This system leverages the ESP32 microcontroller and an RS485-based soil sensor to provide accurate data on soil health, enabling informed agricultural decisions.

The project focuses on delivering a low-cost, scalable, and efficient monitoring system suitable for precision agriculture and smart farming applications.

🎯 Objectives
To monitor soil parameters in real time
To enable data-driven agricultural decisions
To build a low-cost IoT solution for farmers
To integrate cloud-based monitoring with a local dashboard

🚀 Features
📡 Real-time soil monitoring
🌐 Local web dashboard hosted on ESP32
☁️ Cloud integration using ThingSpeak
🔌 RS485 Modbus RTU communication
📊 Continuous data logging and visualization

📊 Parameters Monitored
Soil Moisture
Soil pH
Electrical Conductivity (EC)
Nitrogen (N)
Phosphorus (P)
Potassium (K)

🛠️ Hardware Components
ESP32 DevKit C
ZTS 3002 7-in-1 Soil Sensor

Connecting Wires

💻 Software & Technologies
Arduino IDE (Embedded Programming)
Embedded C/C++
HTML, CSS, JavaScript (Web Dashboard)
ThingSpeak (Cloud Platform)

⚙️ System Architecture
The soil sensor collects real-time soil data
Data is transmitted via RS485 using Modbus RTU
ESP32 processes and parses sensor data
Data is:
Displayed on a local web server dashboard
Uploaded to ThingSpeak cloud platform
Users can monitor soil conditions remotely
